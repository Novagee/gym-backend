什么是Nova Beacon?

Nova Beacon是小型无线传感器，用户将Nova Beacon连接到任何位置和物品。当任何智能手机进行Nova Beacon设定区域时，可自动接收信息。Nova Beacon可识别设定区域内的位置地点，温度，运动路线，和物品类型所属权。

通过新星科技的设定，用户可用智能手机上的应用软件分析设定Nova Beacon所收集的信息。

新星科技把应用程序和Nova Beacon结合，把世界连接你的智能设备。

Nova Beacon是如何运行的

Nova Beacon是一个迷你电脑。他拥有强大的ARM处理器，内存，蓝牙智能模块，和温度，运动传感器。

Nova Beacon和应用软件构建一个移动交互平台。

新星科技可在零售、教育、医疗、交通和酒店等行业构建应用程序。
